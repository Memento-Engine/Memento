## What is Memento AI?

**Memento AI** is a privacy-first desktop application that indexes everything on your machine — files, notes, documents, browser history — and lets you retrieve it with natural language queries in milliseconds.

Unlike cloud-based tools, **your data never leaves your device**. Memento runs a Rust-powered daemon locally, builds a hybrid semantic + keyword index in real time, and optionally routes through a local AI Gateway for synthesised, cited answers.

## How it works

```
    Your Files, Notes & History
               │
               ▼
    ┌──────────────────────┐
    │   Rust Daemon        │  watches folders, real-time indexing
    │   (crates/daemon)    │
    └───────────┬──────────┘
                │  local hybrid index
                ▼
    ┌──────────────────────┐
    │   Rust Core          │  semantic + keyword search, sub-ms
    │   (crates/core)      │
    └───────────┬──────────┘
                │  optional AI augmentation
                ▼
    ┌──────────────────────┐
    │   AI Gateway         │  Node.js · auth · rate-limiting
    │   (ai-gateway/)      │  routes to OpenAI or local LLM
    └───────────┬──────────┘
                │
                ▼
    ┌──────────────────────┐
    │   Desktop UI         │  Next.js 16 + React 19 + Tauri
    │   (app/frontend/)    │
    └──────────────────────┘
```

1. **Index** — The daemon watches your configured folders and continuously rebuilds the index.  
2. **Search** — Queries hit the local index first — results in under a millisecond.  
3. **Augment** — Optionally send a query to the AI Gateway for a cited, synthesised answer.  
4. **Display** — Results surface in the Tauri desktop app or the local Next.js UI.

## Tech stack at a glance

| Layer | Technology |
|---|---|
| Desktop shell | Tauri 2 |
| Frontend UI | Next.js 16 · React 19 · Tailwind CSS · Radix UI |
| Agent logic | LangChain · LangGraph |
| AI Gateway | Node.js · Express · TypeScript |
| Database | PostgreSQL · Drizzle ORM |
| Search core | Rust (`crates/core`) |
| Background daemon | Rust (`crates/daemon`) |
| Observability | Sentry · Pino |
