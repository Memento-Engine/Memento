## [1.0.0] - 2024-03-24

<Note>
  **Initial stable release** - Production-ready with complete feature set.
</Note>

### ✨ Features

- **Continuous Screen Capture**: Automatic screen recording with adaptive throttling
- **OCR Processing**: Windows OCR API integration with 25+ language support
- **Semantic Search**: Vector search using ONNX Runtime with DirectML GPU acceleration
- **Hybrid Search**: Combined semantic + keyword search with reciprocal rank fusion
- **AI Chat**: GPT-4o and Claude integration with conversation memory
- **Privacy Controls**: Application and website exclusions with retroactive filtering
- **Auto-Updates**: Velopack integration with delta patches (90%+ size reduction)
- **Windows Service**: 24/7 background operation with automatic startup
- **System Tray**: Minimal UI for quick access
- **Dark Mode**: Vercel-inspired design system

### 🏗️ Architecture

- **Rust Daemon**: Core capture and indexing engine
- **AI Gateway**: Node.js/Express API with PostgreSQL
- **Next.js Frontend**: React 19 with Tailwind CSS
- **Tauri Desktop**: Cross-platform wrapper with IPC
- **SQLite Database**: Local data storage with sqlite-vec
- **ONNX Models**: Sentence transformers for embeddings

### 📊 Performance

- **Capture Interval**: 2-5 seconds (adaptive)
- **OCR Latency**: <100ms per frame
- **Search Speed**: <50ms for semantic search
- **Database Size**: ~100MB per 10,000 frames
- **Memory Usage**: ~200MB idle, ~500MB active

### 🔒 Security

- **Local-First**: All data stored locally by default
- **Privacy Masking**: Exclude sensitive apps/sites
- **OAuth Integration**: Google Sign-In for gateway
- **Rate Limiting**: Token bucket + sliding window

### 📦 Deployment

- **Installer Size**: 85 MB (full), 3-8 MB (delta updates)
- **Supported OS**: Windows 10/11 (64-bit)
- **Requirements**: 4GB RAM, 2GB disk space
- **GPU**: Optional DirectML acceleration

## [0.9.0] - 2024-03-10

### 🎯 Beta Release

- ✅ All core features implemented
- ✅ Internal testing complete
- ✅ Documentation complete
- ⚠️ Known issues with service stability

### Added

- Windows Service integration
- Auto-update system
- Rate limiting in AI Gateway
- Usage tracking and analytics
- Session management

### Fixed

- Service crash on Windows shutdown
- Memory leak in OCR pipeline
- Database lock issues
- Port conflict resolution

### Changed

- Switched from Actix to Axum for HTTP server
- Migrated from OpenAI SDK to direct API calls
- Updated Tauri to v2.0
- Redesigned system tray UI

## Migration Guides

### 0.9.x → 1.0.0

No breaking changes. Update automatically via built-in updater.

**Optional**: Reindex for improved search quality
```powershell
.\memento-daemon.exe reindex
```

### 0.8.x → 0.9.0

**Breaking**: Database schema change requires migration

<Steps>
  <Step title="Backup database">
    ```powershell
    Copy-Item $env:APPDATA\Memento\memento.db $env:APPDATA\Memento\memento.db.bak
    ```
  </Step>
  
  <Step title="Run migration">
    ```powershell
    .\memento-daemon.exe migrate --from 0.8.0 --to 0.9.0
    ```
  </Step>
  
  <Step title="Verify">
    ```sql
    sqlite3 memento.db "SELECT COUNT(*) FROM chunks;"
    ```
  </Step>
</Steps>

## Semantic Versioning

Memento AI follows [Semantic Versioning 2.0.0](https://semver.org/):

**MAJOR.MINOR.PATCH**

- **MAJOR**: Breaking changes (database schema, API changes)
- **MINOR**: New features (backward compatible)
- **PATCH**: Bug fixes (backward compatible)

**Examples**:
- `1.0.0 → 1.0.1`: Bug fix (auto-update)
- `1.0.1 → 1.1.0`: New feature (auto-update)
- `1.1.0 → 2.0.0`: Breaking change (manual migration required)
