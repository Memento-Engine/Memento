## Overview

Memento AI respects your privacy. This guide shows you how to configure exclusion rules for sensitive websites and applications.

---

## Quick Start

### Exclude Banking Websites

```bash
curl -X POST http://localhost:7070/privacy/masked \
  -H "Content-Type: application/json" \
  -d '{
    "name": "bank.com",
    "item_type": "web"
  }'
```

### Exclude Password Manager

```bash
curl -X POST http://localhost:7070/privacy/masked \
  -H "Content-Type: application/json" \
  -d '{
    "name": "KeePass.exe",
    "item_type": "app"
  }'
```

## Retroactive Cleanup

After adding exclusions, run privacy refresh to delete existing data:

```bash
curl -X POST http://localhost:7070/privacy/refresh
```

<Warning>
  This permanently deletes matching frames and chunks. Cannot be undone.
</Warning>
