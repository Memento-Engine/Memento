## Base URLs

| Service | Base URL | Description |
|---|---|---|
| AI Gateway | `http://localhost:4000` | LLM routing, auth, usage |
| Rust Daemon | `http://localhost:5100` | Local index control |

All requests and responses use `application/json`.

---

## Authentication

Most AI Gateway endpoints require a valid session cookie. Obtain one by logging in:

```bash
POST http://localhost:4000/auth/login
Content-Type: application/json

{
  "email": "you@example.com",
  "password": "..."
}
```

The response sets a `connect.sid` session cookie. Include it in subsequent requests.
